How to fix authentication

YouTube requires authentication. To download videos, you need to provide a `cookies.txt` file.

  1. Install the "Get cookies.txt LOCALLY" extension in your browser (Chrome/Edge).
  2. Go to youtube.com and log in.
  3. Click the extension icon and click "Export".
  4. Save the file as cookies.txt in the project root folder.
  5. Restart the server.
